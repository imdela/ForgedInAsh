## Workflow de Génération Recommandé

```
1. OUVRIR  `03_generation_prompts.md`
2. COPIER  le PROMPT SYSTÈME PRINCIPAL
3. AJOUTER le CONTEXT BLOCK complété avec l'état actuel
4. GÉNÉRER la scène
5. VÉRIFIER avec la SCENE REVIEW CHECKLIST
6. SI score estimé > 25% → utiliser le PROMPT DE RÉVISION ANTI-IA
7. VALIDER et archiver
```

## Prochaine Action Requise

> [!IMPORTANT]
> Le fichier `05_personal_style_notes.md` est **vide et en attente**.
> C'est le fichier le plus important du dossier — il contient ta voix propre.
> Il doit être rempli par toi avec :
>
> - Des exemples de ta propre écriture (textes, emails longs, essais)
> - Tes tics lexicaux naturels
> - Les métaphores que tu utilises naturellement
> - Les sujets/images vers lesquels tu reviens sans le décider
> - Les structures de phrase que tu préfères instinctivement

Sans ce fichier, le style Vera Draye reste une synthèse des autres autrices.
Avec ce fichier, il devient singulier.
